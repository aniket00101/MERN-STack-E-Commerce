# 🛒 NextMart – MERN E-Commerce Platform

Live at:- https://ecommercefrontenddeploy-one.vercel.app/
# 
## 🔹 Overview

- This is a full-featured E-Commerce Web Application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js) with a modern UI powered by Tailwind CSS.
- The platform allows users to browse products, manage carts, place orders, and make secure online payments via PayPal Integration.
- It also includes a powerful admin dashboard for managing products, users, and orders.

## ✨ Features

- 🛍️ Product Browsing – View products with categories, filters, and search.
- 🔎 Advanced Filtering – Filter by price, category, size, brand, and more.
- 👤 User Authentication – Secure Login & Registration using JWT.
- 🛒 Smart Cart System – Add, update, and remove products from cart.
- 💳 PayPal Payment Integration – Secure online checkout.
- 📦 Order Management – Track orders and view order history.
- 🛠️ Admin Dashboard – Manage products, users, and orders.
- 📱 Fully Responsive – Optimized for mobile, tablet, and desktop.
- ⚡ Fast Performance – Optimized frontend and backend deployment.
- 🔐 Protected Routes – Role-based authentication (User/Admin).


## 🚀 Tech Stack
# Frontend: 
- React.js – Component-based UI development.
- Vite – Fast build tool.
- Tailwind CSS – Utility-first modern styling.
- Redux Toolkit – State management.
- Axios – API communication.
- React Router – Routing system.

# Backend
- Node.js – JavaScript runtime.
- Express.js – REST API framework.
- MongoDB Atlas – Cloud database.
- Mongoose – ODM for MongoDB.
- JWT – Authentication & authorization.
- PayPal API – Online payment gateway.

# Deployment
- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas

## 🚀 How It Works
- User registers or logs in.
- Backend generates a JWT token for authentication.
- User browses products and adds items to cart.
- At checkout, user selects PayPal payment.
- Payment is securely processed via PayPal API.
- After successful payment, order is stored in MongoDB.
- Admin can manage products, users, and orders from dashboard.
